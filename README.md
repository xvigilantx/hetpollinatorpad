# Het Pollinatorpad

Website voor Het Pollinatorpad - een 30 kilometer lang natuurpad door de Betuwe.

## Over het project

Het Pollinatorpad is een initiatief van The Pollinators en Gonzend Rivierenland om een bloeiend lint van biodiversiteit te creëren door de Betuwe. De route verbindt verschillende dorpen en landschappen, met als doel een doorlopende corridor te creëren voor bestuivers.

## Website structuur

```
/
├── index.html          # Hoofdpagina
├── images/            # Foto's en afbeeldingen
│   ├── kaart.png      # Routekaart
│   └── ...            # Bestuiver en bloemen foto's
└── assets/            # Video en andere media
    └── Hommel__7_.mp4 # Hero video
```

## Features

- Video achtergrond met hommel in de hero sectie
- Custom SVG logo en illustraties
- Responsive design
- Interactieve progress bar
- Foto galleries van bestuivers en community activiteiten
- Routekaart van Het Pollinatorpad

## Deployment

Deze website kan gehost worden op:
- GitHub Pages
- Netlify
- Vercel
- Elke static hosting provider

Voor GitHub Pages:
1. Push naar GitHub repository
2. Ga naar Settings > Pages
3. Selecteer branch (main/master)
4. Website is beschikbaar op `username.github.io/repository-name`
